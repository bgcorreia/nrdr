NRDR - The Non-coding RNA Databases Resource -- Release 1.1
-------------------------------------------

1. SUMMARY

NRDR - The Non-coding RNA Databases Resource -- Release 1.1 of the database contains 9 new databases and 3 new databases update. This new release contains now 111 databases (the previous release had 102 databases).
The NRDR is a website freely available to all through the web interface at http://www.ncrnadatabases.org/.

Release 1.1 - November/December - 2012

=================================
