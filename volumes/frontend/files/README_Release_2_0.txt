NRDR - The Non-coding RNA Databases Resource -- Release 2
-------------------------------------------

1. SUMMARY

NRDR - The Non-coding RNA Databases Resource -- Release 2 integrates all the update comes from version 1.0, 1.1 and 1.2 and more 21 new databases. Now in version 2.0 contains a total of 140 databases.
The NRDR is a website freely available to all through the web interface at http://www.ncrnadatabases.org/.
