NRDR - The Non-coding RNA Databases Resource -- Release 1.2
-------------------------------------------

1. SUMMARY

NRDR - The Non-coding RNA Databases Resource -- Release 1.2 of the database contains 8 new databases. This new release contains now 119 databases (the previous release 1.1 had 111 databases).
The NRDR is a website freely available to all through the web interface at http://www.ncrnadatabases.org/.
This release contains databases for long non-coding RNAs.
